# CoffeeGuard

Скрипт на языке Python для автоматизации бизнес-аналитики и контроля издержек в точках общественного питания (кофейнях). 

Проект осуществляет парсинг сменных отчетов, рассчитывает операционные показатели продаж напитков и вычисляет KPI утилизации сырья.

## Основной функционал
- **Анализ объемов продаж:** Агрегация данных по проданным позициям (Латте, Капучино, Флэт Уайт) за отчетный период.
- **Расчет финансовой аналитики:** Оценка общей выручки на основе зафиксированных цен за единицу продукции.
- **Контроль издержек (KPI):** Расчет процентного соотношения утилизированного молока к общему объему сырья для мониторинга эффективности работы персонала.

## Стек технологий
- Python 3.13
- Pandas 

## Входные данные
Скрипт обрабатывает данные из файла `coffee_data.csv`, содержащего следующие метрики:
- `date` — дата отчетных суток
- `latte`, `cappuccino`, `flat_white` — количество проданных чашек по категориям
- `milk_start`, `milk_end`, `milk_waste` — баланс и утилизация молока (в литрах)
